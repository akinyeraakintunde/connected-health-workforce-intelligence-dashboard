# Connected Health Workforce Intelligence Dashboard

## Project Overview

This project demonstrates the design and development of a Workforce Intelligence Dashboard using Power BI for a healthcare provider environment.

The dashboard provides operational visibility into workforce capacity, care delivery performance, employee absence, training compliance, and workforce costs.

## Business Problem

Healthcare providers require accurate workforce data to:

- Monitor staffing levels
- Improve care delivery
- Reduce missed visits
- Track compliance
- Control overtime costs
- Support workforce planning

## Solution

A Power BI dashboard was developed to provide:

- Executive workforce reporting
- Care visit analytics
- Staff utilisation monitoring
- Training compliance reporting
- Absence and retention insights
- Workforce cost analysis

## Tools Used
